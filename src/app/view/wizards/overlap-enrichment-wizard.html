We are going to perform the Locus Overlap Enrichment Analysis (LOLA) (Sheffield and Bock 2016).

The background is used to define which regions are basis of the analysis. Datasets are used for enriching the query regions.
Overlaps between the background regions and query regions are used to redefine the query regions and then count the overlaps between the query and each dataset regions. With these values, a contingency matrix is build, and the Fisher exact test applied to it.
